# widget_bank

## Цель проекта: 
освоить элементарные команды для работы с GIT


## Описание:

Главный файл для запуска: *src/widget.py*
В файле собран запуск всех имеющихся модулей, результат выводится в консоли


## Инструкция для запуска из консоли

Скопируйте и в консоли выполните команду : 

```bash
python3.12 src/widget.py
```

## Тесты 
Добавлены unit-тесты для всех функций проекта.
Тестирование выполняется с помощью фреймворка <span style="font-size: 120%; color: green; font-family: monospace;">pytest</span>
Дополнительно установлена библиотека **pytest-cov** для оценки процента тестирования

Запуск тестов выполняется командой:
```bash
pytest
```
